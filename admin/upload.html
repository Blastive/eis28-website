How to use

  1. Enter your GitHub token once (it saves in your browser)
  2. Select the event category from the dropdown
  3. Drag photos into the box, or click it and select multiple files (Ctrl+click or drag-select in the file picker)
  4. Click "Upload All Photos"
  5. Wait for all photos to upload — they'll appear on the website automatically

Photos are uploaded to images/{category}/ in your GitHub repo. The website reads them automatically.
